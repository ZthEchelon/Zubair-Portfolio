# Zubair Portfolio

A modern portfolio website built with React, TypeScript, and Vite.

## Development

To run the project locally:

```bash
npm install
npm run dev
```

## Building

To build the project for production:

```bash
npm run build
```

## Deploying to GitHub Pages

This project is configured to deploy automatically to GitHub Pages.

### Deploy the site

Run the following command to build and deploy to GitHub Pages:

```bash
npm run deploy
```

This will:
1. Build the client application (using `vite build`)
2. Deploy the built files to the `gh-pages` branch
3. Publish to `https://ZthEchelon.github.io/Zubair-Portfolio`

### Enable GitHub Pages (First-time setup)

After your first deployment, you need to configure GitHub Pages in your repository settings:

1. Go to your repository on GitHub
2. Navigate to **Settings** > **Pages**
3. Under **Source**, select the `gh-pages` branch
4. Click **Save**
5. Your site will be published at `https://ZthEchelon.github.io/Zubair-Portfolio`

Note: It may take a few minutes for your site to become available after the first deployment.
